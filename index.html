About Stock Globe Trotter

User Story

AS AN Investor who travels I WANT to keep track of my stock's exchange rate for where I am SO I know my stock's value in relation to my location.

Technologies Used

The two APIs we used in this project are the NASDAQ Data API and ExchangeRates API
